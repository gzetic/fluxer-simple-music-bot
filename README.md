# fluxer-simple-music-bot
This is a simple bot for fluxer which can play music from Youtube

## Usage
### Commands:
*/play <song name>*: to play a song from Youtube

*/stop* or */pause*: to pause music playback

*/resume* or */play* to resume music playback

*/leave*: Get the bot to leave the voice channel

*/help*: to display this list of commands

## Installation
Prerequisites: Node

```npm install debug```
```npm install```

Make your bot in fluxer, get the bot token and use it in the place of "your_token":

```FLUXER_BOT_TOKEN=your_token node bot.js```
